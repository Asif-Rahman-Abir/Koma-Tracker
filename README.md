# Koma Tracker

A premium, high-performance web application to track Anime, Manga, and Manhwa using the AniList GraphQL API.

## Features

- **Premium UI**: Dark mode, glassmorphism, and smooth animations.
- **Trending & Popular**: Real-time data from AniList.
- **Series Analysis**: Detailed breakdown of manga series, including relations (prequels/sequels).
- **Manga Tracking**: Track your reading progress (Volumes/Chapters).

## Tech Stack

- **Framework**: React + Vite + TypeScript
- **Styling**: Tailwind CSS
- **State Management**: TanStack Query
- **Backend/Auth**: Supabase
- **API**: AniList GraphQL

## Getting Started

1.  Clone the repository.
2.  Run `npm install`.
3.  Create a `.env` file with your Supabase keys (see `implementation_plan.md` in docs).
4.  Run `npm run dev`.
